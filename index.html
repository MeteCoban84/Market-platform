How this is built · who builds these

Leading open-source comparable: Ghostfolio by Thomas Kaul (~8k★, AGPL-3.0): Angular + NestJS + Postgres + Redis, fetching Yahoo/Alpha Vantage/CoinGecko on a schedule. rotki is the privacy-first, local-encrypted alternative. Lesson both teach: serious tools run a tiny backend that fetches + caches — browsers cannot fetch live equities reliably.

Data reality 2026: equities have no reliable keyless browser feed (Yahoo deprecated, Stooq now key-gated). CoinGecko (crypto) and ExchangeRate-API / Frankfurter (FX) are keyless + CORS-open — so this app does those live, and routes equities through chat.

Inside the iOS Claude app the network is sandboxed, so feeds fall back to the snapshot. Open in Safari/Chrome for live crypto + FX. Not investment advice.
